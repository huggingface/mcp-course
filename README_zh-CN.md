## 课程内容

本课程分为 4 个单元。这些单元将带你从 **模型上下文协议 (Model Context Protocol, MCP) 的基础知识逐步深入到在 AI 应用中实现 MCP 的最终项目**。

在此注册（完全免费）👉 [即将推出]

访问课程 👉 [即将推出]

| 单元 | 主题 | 描述 |
| ------- | --------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 0 | 欢迎来到课程 | 欢迎词、课程指南、必要工具和课程概览。 |
| 1 | Model Context Protocol 介绍 | MCP 的定义、核心概念及其在连接 AI 模型与外部数据和工具中的作用。 |
| 2 | 使用 MCP 进行开发：实践应用 | 学习如何使用现有 SDK 和框架实现 MCP 客户端和服务器。 |
| 3 | MCP 协议深度解析 | 探索高级 MCP 功能、架构和实际集成模式。 |
| 4 | 额外单元与合作项目 | 特殊主题、合作库和社区驱动的项目。 |

## 前置要求

* 对 AI 和 LLM 概念有基本了解
* 熟悉软件开发原理和 API 概念
* 至少掌握一种编程语言（课程将重点展示 Python 和 TypeScript 示例）
