## 您将构建什么

**PR Agent 工作流服务器** —— 一个能让 Claude Code 变得“团队感知”和“工作流智能”的 MCP 服务器：

- **智能 PR 管理**：运用 MCP 工具，根据代码变更自动推荐最合适的 PR 模板。
- **CI/CD 监控**：借助 Cloudflare Tunnel 和标准化的提示，实时追踪 GitHub Actions 状态。
- **高效团队沟通**：通过 Slack 通知，展示 MCP 三大核心能力如何无缝协同，打通团队信息流。

## 真实场景案例

我们将聚焦于每个开发团队都会面临的日常挑战，并展示如何通过 MCP 解决它们：

**优化前**：开发者手动创建 PR，焦急地等待 Actions 完成，然后手动检查结果，并需要手动通知相关团队成员。整个过程繁琐且易出错。

**优化后**：连接到您的工作流服务器后，Claude Code 能够智能地：
- 根据变更的文件，精准建议合适的 PR 模板。
