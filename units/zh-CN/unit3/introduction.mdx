## 你将构建什么

**PR Agent 工作流服务器** - 一个 MCP 服务器，演示如何让 Claude Code 具备团队感知和工作流智能：

- **智能 PR 管理**：使用 MCP 工具根据代码变更自动选择 PR 模板
- **CI/CD 监控**：通过 Cloudflare Tunnel 和标准化提示词跟踪 GitHub Actions
- **团队沟通**：Slack 通知演示所有 MCP 原语协同工作
