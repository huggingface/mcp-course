# 欢迎来到 🤗 模型上下文协议（MCP）课程

![MCP 课程缩略图](https://huggingface.co/datasets/mcp-course/images/resolve/main/unit0/1.png)

欢迎来到当今人工智能领域最激动人心的主题：**模型上下文协议（Model Context Protocol，MCP）**！

这门免费课程与 [Anthropic](https://www.anthropic.com) 合作开发，将带你踏上一段旅程，**从初学者到精通**，理解、使用并构建基于 MCP 的应用程序。

第一单元将帮助你入门：

* 了解**课程大纲**。
* **获取有关认证流程和时间安排的更多信息**。
* 认识课程背后的团队。
* 创建你的**账户**。
* **注册我们的 Discord 服务器**，结识你的同学和我们。

让我们开始吧！

## 这门课程能带给你什么？

在本课程中，你将：

* 📖 从**理论、设计和实践**角度学习模型上下文协议。
* 🧑‍💻 学习**使用成熟的 MCP SDK 和框架**。
* 💾 **分享你的项目**并探索社区创建的应用程序。
* 🏆 参与挑战，**将你的 MCP 实现与其他学生的实现进行评估对比**。
* 🎓 通过完成作业**获得结业证书**。

还有更多！

在本课程结束时，你将理解 **MCP 的工作原理，以及如何使用最新的 MCP 标准构建利用外部数据和工具的 AI 应用程序**。

别忘了[**注册课程！**](https://huggingface.co/mcp-course)

## 课程是什么样的？

课程由以下部分组成：

* _基础单元_：在这里你将学习 MCP 的**理论概念**。
* _实践操作_：在这里你将学习**使用成熟的 MCP SDK** 来构建你的应用程序。这些实践部分将提供预配置的环境。
* _用例作业_：在这里你将应用所学概念来解决你选择的实际问题。
* _合作项目_：我们与 Hugging Face 的合作伙伴合作，为你提供最新的 MCP 实现和工具。
 
这门**课程是一个不断发展的项目，会根据你的反馈和贡献而演进！**欢迎在 GitHub 上提出问题和 PR，并在我们的 Discord 服务器上参与讨论。

## 有哪些先决条件？

要能够学习本课程，你应该具备：

* 对 AI 和 LLM 概念的基本理解
* 熟悉软件开发原则和 API 概念
* 至少掌握一种编程语言的经验（课程将展示 Python 或 TypeScript 示例）

如果你不具备这些条件，别担心！以下是一些可以帮助你的资源：

* [LLM 课程](https://huggingface.co/learn/llm-course/)将指导你了解使用和构建 LLM 的基础知识。
* [智能体课程](https://huggingface.co/learn/agents-course/)将指导你使用 LLM 构建 AI 智能体。

> [!TIP]
> 上述课程本身不是先决条件，所以如果你理解 LLM 和智能体的概念，你现在就可以开始本课程！

你只需要 2 样东西：

* 一个_账户_：用于访问课程资源和创建项目。如果你还没有账户，可以在[这里](https://huggingface.co/join)创建一个（免费）。

## 认证流程

你可以选择以_旁听模式_学习本课程，或者完成活动并_获得我们将颁发的两种证书之一_。如果你旁听课程，你可以参与所有挑战并完成作业（如果你愿意），**你不需要通知我们**。

认证流程是**完全免费的**：

* _获得基础认证_：你需要完成课程的第 1 单元。这适用于想要了解 MCP 最新趋势但不需要构建完整应用程序的学生。
* _获得结业证书_：你需要完成用例单元（第 2 和第 3 单元）。这适用于想要构建完整应用程序并与社区分享的学生。
