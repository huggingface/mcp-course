# 欢迎学习 🤗 模型上下文协议 (MCP) 课程

![MCP 课程缩略图](https://huggingface.co/datasets/mcp-course/images/resolve/main/unit0/1.png)

欢迎前来探索当今人工智能领域最激动人心的主题：**模型上下文协议 (Model Context Protocol, MCP)**！

本免费课程由我们与 [Anthropic](https://www.anthropic.com) 联合开发，将带您**从入门到精通**，深入理解和使用 MCP，并亲手构建基于 MCP 的应用程序。

在第一单元，我们将帮助您：

* 了解**课程大纲**
* **了解认证流程与时间安排**
* 认识课程开发团队
* 创建您的**学习账户**
* **加入我们的 Discord 服务器**，与同学和老师们交流

让我们开始吧！

## 您将收获什么？

在本课程中，您将：

* 📖 从**理论、设计及实践**全方位学习模型上下文协议。
* 🧑‍💻 学习**如何使用主流的 MCP SDK 与框架**。
* 💾 **分享您的项目**，并探索社区成员创建的精彩应用。
* 🏆 参与挑战，**将您的 MCP 实现与其他同学的实现进行切磋、评估**。
* 🎓 完成所有作业，**赢取结业证书**。

还有更多精彩等待您！

课程结束后，您将**掌握 MCP 的工作原理，并能运用最新的 MCP 标准来构建可调用外部数据及工具的 AI 应用**。

别忘了[**报名课程！**](https://huggingface.co/mcp-course)

## 课程形式

课程包含以下几个部分：

* _理论基础_：学习 MCP 的**核心理论**。
* _动手实践_：学习**使用主流 MCP SDK** 构建应用。我们为您准备了预置好的开发环境。
* _项目作业_：运用所学知识解决您感兴趣的实际问题。
* _业界合作_：我们与 Hugging Face 的合作伙伴携手，为您带来最前沿的 MCP 实现与工具。
 
本**课程将持续迭代，并根据您的反馈与贡献不断完善！**欢迎您在 GitHub 上提出问题和 PR，或在我们的 Discord 服务器上参与讨论。

## 学习先决条件

为顺利完成本课程的学习，您应具备：

* 对 AI 和大语言模型 (LLM) 概念有基本了解
* 熟悉软件开发基本原则及 API 概念
* 至少掌握一种编程语言（课程示例将使用 Python 或 TypeScript）

如果您暂时不具备这些条件，也请不必担心！以下资源可以帮助您快速入门：

* [LLM 课程](https://huggingface.co/learn/llm-course/)将带您入门 LLM 的使用与构建。
* [智能体课程](https://huggingface.co/learn/agents-course/)将带您学习如何使用 LLM 构建 AI 智能体。

> [!TIP]
> 上述课程并非硬性要求，只要您对 LLM 和智能体有基本概念，就可以立即开始本课程的学习！

## 需要准备什么？

您只需要两样东西：

* 一台可以上网的电脑。
* 一个 **Hugging Face 账户**：用于访问课程资源和创建项目。如果您还没有账户，可以**免费**[在此创建](https://huggingface.co/join)。

## 认证流程

您可以选择以_旁听模式_学习，也可以完成所有课程活动来_获取我们颁发的两种证书之一_。旁听学员同样可以参与所有挑战、完成所有作业，**且无需告知我们**。

认证流程**完全免费**：

* _基础认证_：您需要完成课程的第 1 单元。此认证面向希望了解 MCP 最新趋势、但暂无需求构建完整应用的学员。
* _结业证书_：您需要完成所有项目单元 (第 2、3 单元)。此证书面向希望构建完整应用并与社区分享的学员。
